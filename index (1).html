우리 아이는 함수를 모르는 게 아니라, 왜 특정 고난도 문항에서만 막히는 걸까요?

학부모의 이 질문에 답할 수 있는 건 베테랑 원장의 직관뿐이었습니다. EDGE는 그 직관을, 데이터로 재현합니다. 문제를 공급하는 LMS가 아닌, 학생의 사고 과정 중 어느 지점에서 논리적 단절이 발생했는지 진단하는 IMS입니다.

01 / The Bottleneck

학원 운영의 진짜 병목은
강사 개인의 역량입니다.

// Surface Diagnosis

"삼각함수가 약하다"의 한계

학부모: "그래서 뭘 어떻게 보충하나요?" / 원장: "…삼각함수 문제를 더 풀려야죠." 단원 단위 진단으로는 학부모를 설득할 수 없습니다.

// Retention Risk

"우리 애는 진짜 늘고 있나요?"

3개월차 학부모의 의심에 답할 데이터가 없습니다. "성실히 하고 있어요"라는 정성적 평가로는 재등록 결정의 근거가 되지 못합니다.

// Single Point of Failure

원장 1인에게 묶인 분석 역량

킬러 문항 오답 분석은 원장의 일과를 가장 많이 차지하는 노동입니다. 원생이 늘수록 원장의 시간은 오답지 위에서 사라지고, 다른 누구도 그 일을 대신하지 못합니다.

// The Real Bottleneck

강사 한 명에게 묶인 노하우

학생 한 명을 가장 잘 아는 강사가 그만두는 순간, 누적된 진단·상담 노하우도 함께 사라집니다. 이전·복제 불가능한 개인 자산 위에 학원이 운영됩니다.

02 / Methodology

단원이 아니라,
사고의 연결고리를 진단합니다.

기존 진단 "이 학생은 삼각함수 단원이 약합니다."
EDGE 진단 삼각방정식 해의 개수 문항에서, 그래프의 주기와 대칭성을 활용하는 논리적 연결이 누락됨. → 예상 보완 기간 14일 / 예상 점수 상승 +8점
— 01

Semi-Micro 진단 입자도

"단원이 약하다"가 아닌, 풀이 논리의 정확한 단절 지점을 짚습니다. 학부모에게 추상적 지적이 아닌 구체적 좌표를 제공합니다.

10s 분석 소요 시간
— 02

풀이 논리 데이터베이스

2022~2026학년도 평가원·교육청 기출 전 문항을 풀이 논리 단위로 해체하여 구축. 단순 정답 매칭이 아닌 사고 경로 매칭.

5y 기출 데이터 범위
— 03

상담 스크립트 자동 출력

오답 번호 입력 → 약점 지점 + 학부모용 상담 스크립트 즉시 출력. 신입 강사도 베테랑 수준의 상담 가능.

+0 학습 곡선

5개년 기출을 풀이 논리 단위로 해체.
2028 개편 체제 기준으로 재분류.

단순한 OCR이 아닙니다. 각 문항이 요구하는 사고의 단계를 수작업으로 분해하고, 학생의 오답이 어느 단계에서 발생했는지 역추적할 수 있도록 설계했습니다. 분류 체계는 2028학년도 개편 수능 기준에 맞춰, 현재 고2·고3은 물론 향후 입시 변화에도 그대로 사용 가능합니다.

03 / What Changes

기존 시스템과 충돌하지 않습니다.
가장 위에 얹어 사용하십시오.

i.

원장의 업무 고도화

수십 분이 소요되던 기출 오답 분석이 10초로 단축됩니다.

INPUT > 학생 오답 번호
OUTPUT > 핵심 약점 좌표 + 상담 스크립트
TIME > ≈ 10s
ii.

강사 관리 역량의 상향 평준화

신입 강사·조교가 상담을 진행해도, 베테랑 원장과 동일한 수준의 진단을 학부모에게 전달합니다.

BEFORE > 강사별 편차 발생
AFTER > 데이터 기반 일관성 확보
IMPACT > 상담 리스크 ↓
iii.

데이터 기반 학부모 신뢰 확보

PDF 리포트 자동 생성. 약점 보완 시 예상 상승 점수를 수치로 제시하여, 등록·연장의 객관적 명분을 만듭니다.

DELIVERABLE > PDF Report
CONTAINS > 사고 오류 좌표 + 예상 점수 +Δ
USE CASE > 등록 / 연장 상담
